# Поиск изображений по текстовому описанию

Proof of Concept (PoC) для фотохостинга «Со Смыслом»: демонстрационная система поиска изображений по текстовому запросу.

## Описание

Пользователь вводит текстовое описание сцены (например, *«A man is crossing a mountain pass on a metal bridge»*), и система возвращает наиболее релевантное изображение из базы.  
Модель оценивает степень соответствия пары (изображение, текст) и выдаёт оценку от 0 до 1.

## Архитектура

- **Векторизация изображений**: ResNet50 (ImageNet, frozen), эмбеддинг размерности 2048.
- **Векторизация текста**: `sentence-transformers/all-MiniLM-L6-v2`, эмбеддинг размерности 384.
- **Модель совместимости**: LightGBM (лучший результат по RMSE = 0.256).
- **Целевая переменная**: нормализованная экспертная оценка соответствия (шкала 1–4 → 0–1).

## Юридические ограничения

Из датасета удалены все пары, содержащие упоминания детей (слова: `child`, `boy`, `girl`, `kid` и др.).  
При инференсе запросы с такими словами возвращают дисклеймер:
> *This image is unavailable in your country in compliance with local laws.*
